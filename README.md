Файл с данными о маршрутах автобусов Конобеево - Расловлево.
В приложении есть функция, которая обновит данные о маршрутах автоматически.
Если обновлять вручную, то необходимо скопировать этот фаайл в папку:
/sdcard/Android/data/com.tsm.bus_schedule/files/
(иногда она может называться так: /Внутренний общий накопитель/Android/data/com.tsm.bus_schedule/files/)
После перезагрузки приложения, данные будут читаться из этого файла.
Обновление файла BUS_Schedule.json производится регулярно, один раз в месяц!
